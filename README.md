# BIOC

BioC XML format can be used to exchange prepared biomedical corpora and
any accompanying annotations between different research groups and
software platforms.

GOALS are to facilitate: 

+ data exchange 
+ data and tool reuse

BIOC distribution includes a JAVA and C++ Package available from 
http://bioc.sourceforge.net

For more about these see:

+ ReadMe_BioC_C++.txt
+ ReadMe_BioC_Java.txt

See [Public Domain Notice](NOTICE.md)
